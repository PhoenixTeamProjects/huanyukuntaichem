# 07 - Image Guide

## Required Production Assets

- Home hero: industrial fuel additive, chemical lab, production, or packaging image.
- Product list images: 4:3 or 1:1.
- Product detail gallery: consistent product/application photos.
- News cover images: 16:9.
- About images: factory, lab, team, certificates.

## Rules

- Store production image metadata and alt text in Directus.
- Use Next/Image when remote assets are connected.
- Avoid uploading different copies of the same image per language.
- Do not use oversized uncompressed originals on public pages.
