# 04 - Project Spec

## Goal

Build a multilingual foreign trade B2B website for fuel additives and industrial chemical products.

## Current Version Target

Version 0.1 establishes the repository, documentation, frontend routes, CMS boundaries, and VPS deployment skeleton.

## Scope

- Home
- Products
- Product category
- Product detail
- News
- News detail
- Applications
- Service
- About
- Contact
- Inquiry submission route

## Content Ownership

Directus is the source of truth for products, categories, news, applications, pages, SEO, image alt text, and inquiries.
