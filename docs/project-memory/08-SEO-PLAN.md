# 08 - SEO Plan

## Required SEO

Every public page needs:

- unique H1
- SEO title
- meta description
- canonical URL
- Open Graph fields
- hreflang for available language versions
- breadcrumb schema where applicable
- product or article schema where applicable

## Multilingual SEO

Supported locales are `en`, `es`, `ru`, `ar`, `fr`, `pt`, `de`, `id`, `tr`, `fa`.

Untranslated Directus content must use a documented fallback, noindex, or not-generate strategy before production launch.
